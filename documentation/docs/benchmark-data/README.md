# Benchmark Dataset

This directory contains the benchmark datasets collected using **VPNLens**.

- `wireguard.json` – Results from **100 benchmark runs** using the WireGuard deployment.
- `headscale.json` – Results from **100 benchmark runs** using the Headscale deployment.
- `combined-results.json` – Combined dataset containing all benchmark results.

Each benchmark report can also be viewed in the browser using:

`https://vpn.samay15jan.com/results/<token>`

Example:

`https://vpn.samay15jan.com/results/19cb4e66fdc6ea5ae0199a0669a3aace74a8d70b813b9bea`
